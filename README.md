# microRandomPass

Простой скрипт для генерации случайного пароля.

## Библеотки

```sh
pip install argparse
```

## Аргументы

|     Аргумент    |  Описание  |
|:---------------:|------------|
|**-l** *<длина\>*|Длина пароля|
