# microRandomPass

Простой скрипт для генерации случайного пароля.

## Библиотеки

```sh
pip install argparse
```

## Аргументы

|     Аргумент    |  Описание  |
|:---------------:|------------|
|**-l** *<длина\>*|Длина пароля|
|**-i** *<количество итераций\>*|Количество паролей|
|**-a, -c, -n**|Только **символы алфавита**, **символы**, **часла**|
